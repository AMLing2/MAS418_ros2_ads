# ROS2 communication with Beckhoff ADS

Documentation for usage and info can be found here [https://amling2.github.io/MAS418_ros2_ads/](https://amling2.github.io/MAS418_ros2_ads/).

Clone the repo with
```bash
git clone --recurse-submodules https://github.com/AMLing2/MAS418_ros2_ads.git
```
Build ADS package:
```bash
cd ADS
meson build
ninja -C build
```
